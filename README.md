# Video_converter   

## video converter using python    
- used to convert video codec    

Args: 
    video_file: mp4 file path   

## Installation   
```
$ sudo apt-get update   
$ sudo apt-get install ffmpeg   
$ sudo apt-get install python-opencv   
$ pip install opencv-python   
$ pip3 install PythonVideoConverter  

```

### Usage    

$ python3 coverter.py
